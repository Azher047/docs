This page takes you from a fresh API key to real Specter data in three steps. You'll find a company, then pull its full record. Every example uses live endpoints, so you can paste your key and run them as you read.

<Steps>
  <Step title="Set your key">
    Every request needs your key in the `X-API-KEY` header. Grab it from the [API Console](https://app.tryspecter.com/settings/api-console), then keep it handy for the calls below. The base URL for everything is `https://app.tryspecter.com/api/v1`.
  </Step>
  <Step title="Make your first call">
    Start by finding a company by name. This returns the basics, including the Specter ID you'll use to pull more:

    ```bash
    curl "https://app.tryspecter.com/api/v1/companies/search?query=hedgehog" \
      -H "X-API-KEY: YOUR_API_KEY"
    ```

    ```json
    [
      {
        "id": "1a2b3c4d5e6g1a2b3c4d5e6g",
        "name": "Hedgehog Analytics",
        "domain": "hedgehog_analytics.com",
        "founded_year": 2019
      }
    ]
    ```

    If you get this back, your key works and you're talking to the API.
  </Step>
  <Step title="Enrich the company">
    Now pull the full record. Send the domain to the enrich endpoint and you get funding, headcount, location, and more:

    ```bash
    curl -X POST https://app.tryspecter.com/api/v1/companies \
      -H "X-API-KEY: YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{ "domain": "hedgehog_analytics.com" }'
    ```

    ```json
    [
      {
        "id": "1a2b3c4d5e6g1a2b3c4d5e6g",
        "organization_name": "Hedgehog Analytics",
        "hq": { "city": "London", "country": "United Kingdom" },
        "employee_count": 39,
        "funding": {
          "total_funding_usd": 16600000,
          "last_funding_type": "series_a"
        },
        "last_updated": "2026-01-01"
      }
    ]
    ```
  </Step>
</Steps>

That's the core loop: resolve a company, then enrich it. Calls draw on your credit allowance, so check your rate and credit limits before you run at scale.

## Where to go next

Where you head from here depends on your plan.

If you're on the Enrichment API, point it at a list you already have. Send a batch of up to 50 companies or people in one request and append funding, growth, and executive data to your CRM or dealflow.

If you're on the Full API, work from a saved search instead. Build a search in the Specter web app, share it with the API, then pull its matches with `GET /searches/companies/{searchId}/results`. Add `?new=true` to fetch only this week's new matches, and you've got a feed you can poll on a schedule.
