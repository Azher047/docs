title: "LLMs.txt"
description: "Machine-readable documentation for LLMs and AI agents"
---

Specter's documentation is published in formats built for large language models and AI agents, so tools can read the API reference directly.

- [llms.txt](https://api.tryspecter.com/llms.txt) is a structured index of every page in this documentation, with links and short summaries. Point an LLM or agent here first to discover what's available.
- [llms-full.txt](https://api.tryspecter.com/llms-full.txt) is the full text of the documentation in a single file, for tools that want everything in one request.

Both files are generated automatically and stay in sync with the docs.

<Note>
  Every page is also available as Markdown: append `.md` to its URL, for example `https://api.tryspecter.com/api-ref/introduction.md`. Or use the menu at the top of any page to copy it, view it as Markdown, or open it in Claude, ChatGPT, or Perplexity.
</Note>
